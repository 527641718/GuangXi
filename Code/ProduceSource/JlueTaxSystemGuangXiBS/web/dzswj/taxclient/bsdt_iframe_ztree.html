      1、为什么我打开申报表或业务申请表单，界面一片白色没有任何显示？

      答：您需要下载并安装表格控件，表单才能正常显示。

      2、为什么我打开申报表，能看见表单，但全部单元格都显示灰色，不让编辑？

      答：您需要点击IE浏览器上的工具菜单，选择Internet选项-安全-站点-添加，把当前站点加入安全信任站点； 您也可以下载、安装、运行环境检测修复工具，一键修复所有使用环境问题后，重新打开浏览器登录电子税务局。

      3、我如何报送增值税报表？

      答：电子税务局申报操作流程包括：报表填写并保存——申报提交——税费缴纳——报表打印等４个步骤。申报提交功能在我要办税——税费申报及缴纳——税费缴纳——申报表提交模块。 打开该模块后，勾选申报项目名称，点击上方申报数据提交按钮即可报送增值税报表。

      4、一表集成系统没有带出来数据，怎么办？

      答：如发票刚刚上传，请等待增值税发票税控开票软件数据同步后，再进行申报操作；或是可以点击附表1表头上方【删除】按钮，重新点击【刷新发票数据】进行刷新。
